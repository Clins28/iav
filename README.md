# iav.py
可搜索javbus、btso的磁力链接和avgle的预览视频

## 使用方法
*搜索注意按标准书写番号（如BGN-044）*

**注意扶梯子！！**（需要修改代码中的proxy_addr）

在Javbus上搜索（返回字典）iav.py -b 番号

在btso上搜索（返回磁链）iav.py -s 番号

在avgle上搜索预览视频 iav.py -v 番号
